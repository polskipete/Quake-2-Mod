Final Push in Final Branch
Must bind  in config file b4 you start

bind m "teleport"
bind n "flyon"
bind o "fall"

How to play:
'M' is to teleport forward
'N' is to turn on and off gravity
'O' is Emergancy Escape, teleport up

Weapon Changes:
Magic Push (RailGun): Pushes enemies back by a lot
Explosive Potion (Grenade Launcher): Grenade falls down when launched, explodes only on impact
Healing Ray(Machinegun): Heal per second 

Special Weapons and abilites use MANA, which can be spotted next to were ammo use to be
